Project 3

This device replicates the human eye's function by adjusting blinds based on the light intensity detected by sensors. Initially, the window could only be manually operated using a button or remote. However, we enhanced it by integrating ESP32 devices to collect and transmit data to Node-RED. Another ESP32 embedded within the blind uses this data to autonomously control the blinds, enabling automatic adjustments based on light intensity.
